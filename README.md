# Calculadora IMC

## Funcionalidades

- Temas dark e light

- Calcula seu peso e tamanho para dizer qual seu imc, incluindo um alerta se você está abaixo ou acima da média

- Tabela de histórico (feature)

## Stack utilizada

**Front-end:** `Html, Css e Javascript`
